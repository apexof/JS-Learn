1. При объявлении объекта достаточно указать только имя свойства, а значение будет взято из переменной с аналогичным именем.
2. В качестве имени свойства можно использовать выражение
3. Новая функция для проверки равенства значений. alert( Object.is(+0, -0)); // false
4. Object.assign(target, src1, src2...)
Функция Object.assign получает список объектов и копирует в первый target свойства из остальных.
let clone = Object.assign({}, user);
-------------------------------------Методы объекта----------------------------------------------
1. Объявление
let user = {
  sayHi() {
    alert(this.name);
  }
};
2. Также методами станут объявления геттеров get prop() и сеттеров set prop()
3. Вызов super.parentProperty позволяет из метода объекта получить свойство его прототипа.
4. В методах есть специальное внутреннего свойства [[HomeObject]] («домашний объект»), ссылающегося на объект, которому метод принадлежит.
Свойство [[HomeObject]] – не изменяемое.
