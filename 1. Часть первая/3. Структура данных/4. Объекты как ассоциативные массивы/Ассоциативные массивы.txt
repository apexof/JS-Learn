----------------------Создание объектов
o = new Object();
o = {}; // пустые фигурные скобки
Значения которые хранятся в сво-вах объекта – любые значения, любых типов: число, строка – не важно.

----------------------Заполнение сво-вами объекта
литеральный синтаксис:
var menuSetup = {
  width: 300,
  "height": 200,
  title: "Menu"
};

// то же самое, что:

var menuSetup = {};
menuSetup.width = 300;
menuSetup.height = 200;
menuSetup["title"] = 'Menu';

Названия свойств можно перечислять как в кавычках, так и без, если они удовлетворяют ограничениям для имён переменных.

---------------------Удалить св-во из объекта:
delete person.age;

---------------------Проверка св-ва в объекте
if ("name" in obj) { // такую проверку не обманешь искусственным undefined
  alert( "Свойство name существует!" );
}

или
var person = {};
person.lalala === undefined // true, нет свойства с ключом lalala

-------------------------Обращение к св-вам объекта
    через точку:
person.age
    через []
person["age"]

В обоих случаях, имя свойства обязано быть строкой.
Если использовано значение другого типа – JavaScript приведет его к строке автоматически

доступ через переменную
var person = {};
person.age = 25;
var key = 'age';

alert( person[key] ); // выведет person['age']
