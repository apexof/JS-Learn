this это ссылка на объект в контексте которого вызвана функция
this могут использовать только функций
this показывает в каком контексте ВЫЗВАНА функция, не важно где она создана.
this равен объекту перед точкой. Чтобы this передался, нужно вызвать функцию именно через точку (или квадратные скобки).
----------------------------------------------------------------------
Если функция использует this – это подразумевает работу с объектом
function func() {
    alert( this ); // выведет [object Window] или [object global]. А в режиме use strict вместо глобального объекта this будет undefined:
}
func();

--------------------------------------------------------------------
любая операция над результатом операции получения свойства, кроме вызова, приводит к потере контекста.

return this нужен чтобы вернуть ссылку на текущий объект и вызывать методы по цепочке
