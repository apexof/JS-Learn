------------------------------------ Загрузка ----------
DOMContentLoaded – браузер полностью загрузил HTML и построил DOM-дерево. Вешается только через addEventListener.
window.onload (событие load) – браузер загрузил все ресурсы.
   el.onload – если загрузка элемента (картинки, внешнего скрипта <iframe>, <link> (стиля)) успешна.
   el.onerror – если при загрузке произошла ошибка.
window.onunload - человек уходит со страницы или закрывает окно. В нём можно сделать что-то, не требующее ожидания
window.onbeforeunload - если посетитель инициировал переход на другую страницу или нажал «закрыть окно». Может приостановить процесс и спросить подтверждение. ДЛЯ ЭТОГО ЕМУ НУЖНО ВЕРНУТЬ СТРОКУ window.onbeforeunload = function() { return "Данные не сохранены. Точно перейти?";};

------------------------------------ Клавиатура ----------
keydown и keyup - происходят при нажатии/отпускании клавиши
        получить скан-код клавиши в свойстве e.keyCode
keypress - возникает сразу после keydown, если нажатие приводит к появлению символа (буквы, цифры, не Ctrl, Shift).
    получить код символа получить код символа charCode и e.which

------------------------------------ Мышь ----------
Простые события
    selectstart - начало выделения текста
    mousedown - Кнопка мыши нажата над элементом.
    mouseup - Кнопка мыши отпущена над элементом.
    mouseover - Мышь появилась над элементом.
    mouseout - Мышь ушла с элемента.
    mousemove - Каждое движение мыши над элементом генерирует это событие.
    mouseenter/mouseleave похожи на mouseover/mouseout. Отличия: (не учитываются переходы внутри элемента. Не всплывают)
    onscroll - при любой прокрутке и только на прокручиваемых элементах. Событие происходит после прокрутки
    onwheel - генерируется над любым элементом при передвижении колеса мыши. Событие происходит до прокрутки, в нём можно отменить саму прокрутку (действие браузера)
Комплексные события
    click - Вызывается при клике мышью, то есть при mousedown, а затем mouseup на одном элементе. mousedown → mouseup → click
    contextmenu - Вызывается при клике правой кнопкой мыши на элементе.
    dblclick - Вызывается при двойном клике по элементу.

------------------------------------ Формы ----------
focus - пользователь фокусируется на элементе мышкой или Tab
blur – фокус исчезает, например посетитель кликает на другом месте экрана.
focusin/focusout - то же самое, что и focus/blur, только всплывают.
cut/copy/paste - Срабатывают при вставке/копировании/удалении текста.
input - изменение текстовых элементов формы. Не ждет потери фокуса.
change - изменение значения любого элемента формы. Для текстовых элементов срабатывает при потере фокуса.
submit - возникает при отправке формы.

------------------------------------ Др. события ----------
transitionend – когда CSS-анимация завершена.

---------------------------------- Общие св-ва объекта event -------
e.type - Тип события
e.currentTarget - (=this) – элемент, на котором в данный момент сработал обработчик. ПРи делегировании на котором назначен. (до которого «доплыло» событие).
e.target - самый глубокий элемент, на котором произошло событие.
e.eventPhase – на какой фазе он сработал (погружение=1, целе=2 всплытие=3).

---------------------------------- отменить действие браузера -------
1. В обработчике onсобытие вернуть false или event.preventDefault()
    menu.onclick = function(event) { return false; };
2. В обработчике addEventListener только
    event.preventDefault()
