1. Обычно события становятся в очередь и обрабатываются в порядке поступления, асинхронно, независимо друг от друга.
2. Синхронными являются вложенные события (код одного события в другом), инициированные из кода.
      button.onclick = function() {
        text.focus(); // вызов инициирует событие onfocus сразу не дожидаясь конца onclick
      };

3. Чтобы сделать событие гарантированно асинхронным, используется вызов через setTimeout(func, 0).
    button.onclick = function() {
        setTimeout(function() {
          text.focus(); // сработает после onclick, setTimeOut ждет пока выполнится текущая ф-ция.
        }, 0);
    };
