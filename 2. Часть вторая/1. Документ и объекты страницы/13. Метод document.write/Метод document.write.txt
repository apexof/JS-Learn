1. Нет никаких ограничений на содержимое document.write. Строка просто пишется в HTML-документ без проверки структуры тегов
2. Запуск после загрузки приведёт к очистке документа.
3. Иногда document.write используют для добавления скриптов с динамическим URL.

    <script>
      // в url указано текущее разрешение экрана посетителя
      var url = 'http://ads.com/buyme?screen=' + screen.width + "x" + screen.height;

      // загрузить такой скрипт прямо сейчас
      document.write('<script src="' + url + '"></scr' + 'ipt>');
    </script>

    </scr' + 'ipt>' - чтобы браузер не подумал что конец скрипто
