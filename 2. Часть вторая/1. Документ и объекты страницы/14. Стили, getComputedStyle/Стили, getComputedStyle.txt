1. Свойство element.style возвращает объект, который дает доступ к стилю элемента на чтение и запись. element.style.width="100px"
------------------------------------------------ Запись стилей -----------------------------------------------
    1. нужно указывать единицы измерения, например px
    2. Префикс в имени дает заглавную букву background-color  => elem.style.backgroundColor
        1. Исключения: не elem.style.float, а elem.style.cssFloat.
    3. сбросить поставленный стиль, присваивают в style пустую строку: elem.style.width="". Будет взято значение их CSS
    4. Стиль в style находится в формате браузера, а не в том, в котором его присвоили.
        document.body.style.color = '#abc';
        alert( document.body.style.color ); // rgb(170, 187, 204)
    5. Свойство style мы используем лишь там, где не работают классы. например если точное значение цвета/отступа/высоты вычисляется в JavaScript.
    6. Свойство style является специальным объектом, ему нельзя присваивать строку.
    7. чтобы поставить сразу много стилей элементу можно использовать:
        1.el.setAttribute("style", ...)
        2. el.style.cssText = "str"
            При установке style.cssText все предыдущие свойства style удаляются.
------------------------------------------------ Чтение стилей -----------------------------------------------
1. Свойство style содержит лишь тот стиль, который указан в атрибуте элемента, без учёта каскада CSS.
2. Для того, чтобы получить текущее используемое значение свойства, используется метод window.getComputedStyle.
    getComputedStyle(elem).color
    1. Возвращаются именно окончательные значения. При этом все размеры приводятся к пикселям.
    2. getComputedStyle требует полное свойство! Например: paddingLeft, marginTop, borderLeftWidth.
        При обращении к сокращенному: padding, margin, border – правильный результат не гарантируется. Возможны глюки в разных браузерах.
    3. Стили посещенных ссылок – тайна!  не дает доступ к этой информации, чтобы произвольная страница не могла определить, посещал ли пользователь ту или иную ссылку.
