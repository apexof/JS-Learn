-----------------------------------------------window-------------------------------------------
1. window содержит свойства и методы для управления окном браузера.
2. window является глобальным объектом в JavaScript содержащим:
    1. DOM - Глобальный объект document. Его свойства и методы позволяют взаимодействовать с содержимым страницы.
    2. BOM - Объектная модель браузера. Это объекты для работы с чем угодно, кроме документа.
