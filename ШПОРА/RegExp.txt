-------------------------------- Функции поиска по Рег
str.search(reg) - 
  Он возвращает позицию первого совпадения или -1, если ничего не найдено.
  всегда ищет только первое совпадение, флаг g не работает
let result = str.match(reg) - 
  если совпадений не было, возвращает null
  БЕЗ ФЛАГА g (найдет первое совпадение)
    result.input - все строка в которой велся поиска
    result.index - позиция совпадения
    result[0] - всё совпадение полностью
    result[1] - часть совпадения, внутри скобок в регексп, если были
  С ФЛАГОМ g (найдет все совпадения)
    возвращает обычный массив из всех совпадений
str.split(reg) - разобьёт строку по всем совпадениям в массив
  '12-34-56'.split(/-/) ---> [12, 34, 56]
str.replace(reg, newText) - замена всех вхождений
  '12-34-56'.replace( /-/g, ":" )  ---> 12:34:56
  В строке для замены можно использовать специальные символы:



/myreg/.test(str) - проверяет, есть ли хоть одно совпадение в строке str. Возвращает true/false.
/myreg/.exec(str)
  используют для поиска всех совпадений в цикле
  С ФЛАГОМ g
    возвращает первое совпадение, копия вызова str.match(reg)
  БЕЗ ФЛАГА g
    возвращает первое совпадение и запоминает его позицию в свойстве regexp.lastIndex. Последующий поиск он начнёт уже с этой позиции
    Если совпадений не найдено, то сбрасывает regexp.lastIndex в ноль
    вернёт null, после последнего совпадения
    Можно заставить regexp.exec искать сразу с нужной позиции, если поставить lastIndex вручную
-------------------------------- Флаги
i - Если этот флаг есть, то регэксп ищет независимо от регистра
g - Если этот флаг есть, то регэксп ищет все совпадения
m - Многострочный режим.
